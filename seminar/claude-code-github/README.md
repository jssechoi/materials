# Claude Code × GitHub — 업무 자동화 세미나

비개발자 대상 사내 테크 세미나 자료. **Claude Code와 GitHub를 연동해 업무를 자동화하는 방법**을
11개 챕터(준비 → 연결 → 자동화)로 설명하는 슬라이드 덱이다.

## 파일

| 파일 | 설명 |
|---|---|
| `Claude_Code_x_GitHub_Seminar.html` | **배포용 단일 파일 덱**. 삼성 브랜드 폰트·이미지를 모두 base64로 내장 → 파일 하나로 어디서나 그대로 열린다(외부 폰트·자산 불필요) |

## 조작법

- `←` `→` · `Space` — 슬라이드 이동 (처음/끝에서 순환)
- `M` 또는 우상단 **목차** 버튼 — 전체 슬라이드 목차(클릭해 이동)
- `F` — 전체화면

## 구성 (11챕터)

1. 왜 해야 하는가 · 이점 → 2. 환경 설정(VS Code·Claude Code 설치, 명령어·단축키, Auto/Plan 모드)
→ 3. Claude Code 안에서 → 4. GitHub 연동(commit/push/pull/PR 원리) → 5. 저장소 문서 관리·질의
→ 6. `.md` 기록 원리 → 7. `CLAUDE.md` 활용 → 8. 명령 예시·팁 → 9. 이 문서 제작 과정
→ 10. 작업 흐름 요약 → 11. 마무리

## 비고

- 배포용 PDF(30페이지)가 별도로 있으나, 이 PC의 **NASCA DRM 보호본**이라 사외·비-DRM 환경에서는
  열리지 않으므로 이 공개 저장소에는 **포함하지 않았다**. 어디서나 열람하려면 단일 HTML
  (`Claude_Code_x_GitHub_Seminar.html`)을 사용할 것.
